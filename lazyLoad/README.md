## 介绍
<font color="red">**推荐先下载示例项目**</font><br />
特点:
- 支持自定义动画或图片+css
- 支持添加配置文件
- 支持scroll组件或原生事件
- 支持预加载
- 支持设置加载间隔时间和最少过度动画时间
