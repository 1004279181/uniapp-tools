<img src="https://img.shields.io/badge/version-1.3.1-blue.svg?cacheSeconds=2592000" /><br />

## 介绍

<font color="red">**推荐先下载示例项目**</font><br />
封装 uni-app 的 request 方法，使用方法更简单、方便。<br />
特点：

- 支持 callback 回调函数或者 Promise
- 支持全局、局部的请求和响应拦截器
- 支持添加配置文件
- 支持根据接口自定义处理
- 支持所有 method 方法
