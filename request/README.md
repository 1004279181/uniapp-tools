## 介绍
<font color="red">**推荐先下载示例项目**</font><br />
封装uni-app的request方法，使用方法更简单、方便。<br />
特点：
- 支持callback回调函数或者Promise
- 支持全局、局部的请求和响应拦截器
- 支持添加配置文件
- 支持根据接口自定义处理
