## 介绍

<font color="red">**推荐先下载示例项目**</font><br />

- 支持 Promise
- 支持全局、局部(实例)的请求和响应拦截器
- 支持添加多个、删除拦截器，并在响应拦截器中添加发送请求的 config 数据
- 支持添加配置文件
- 支持所有 method 方法
- 支持 params 参数属性
- 支持文件上传和下载
- 支持重新发送请求(刷新 token)
